С помощью Git программисты и разработчики ориентируются в коде и отслеживают изменения. Git помогает вернуть файлы в исходное состояние и видеть изменения, внеёсённые в определённый период. Разработчик выполняет разные команды(например, commit, push), а все измененения синхронизируются с центральным репозиторием. Git - это система которля версий, а GitHub - онлайн-сервис, по сути социальная сеть.
